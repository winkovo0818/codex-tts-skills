# Codex TTS Skills

Two Codex skills for a novel-to-audio workflow:

- `novel-to-tts-script`: convert novel chapters into clean episode-based TTS narration scripts.
- `tts-script-to-audio`: convert TTS narration scripts into audio files or provider-ready audio generation assets.

Workflow:

```text
novel chapter -> novel-to-tts-script -> TTS script -> tts-script-to-audio -> audio files
```
